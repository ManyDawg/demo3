dnf install -y cups cups-pdf

В /etc/cups/cupsd.conf меняем Listen localhost:631 на Port 631

[Нужен скриншот: изменения Listen на Port 631 в /etc/cups/cupsd.conf]

<Location /> Order allow,deny Allow 192.168.100.0/27 Allow 192.168.200.0/27 Allow 127.0.0.1 </Location>

6. Логирование при помощи rsyslog

Задание: Реализуйте логирование при помощи rsyslog на устройствах HQ-RTR, BR-RTR, BR-SRV: сервер сбора логов расположен на HQ-SRV, убедитесь что сервер не является клиентом самому себе, приоритет сообщений должен быть не ниже warning, все журналы должны находиться в директории /opt, для каждого устройства должна выделяться своя поддиректория которая совпадает с именем машины. Реализуйте ротацию собранных логов на сервере HQ-SRV: ротируются все логи находящиеся в директории и поддиректориях /opt, ротация производится один раз в неделю, логи необходимо сжимать, минимальный размер логов для ротации — 10МБ.

HQ-SRV — сервер логов:

dnf install -y rsyslog

Убираем комментарии (#) перед этими строками в /etc/rsyslog.conf:

#module(load="imudp")

#input(type="imudp" port="514")

#module(load="imtcp")

#input(type="imtcp" port="514")

[Нужен скриншот: раскомментированных модулей imudp и imtcp в /etc/rsyslog.conf]

В конце файла указываем путь для сохранения логов:

$template RemoteLogs,"/opt/%HOSTNAME%/%PROGRAMNAME%.log" *.warning ?RemoteLogs

[Нужен скриншот: добавленного шаблона RemoteLogs в /etc/rsyslog.conf]

mkdir -p /opt

systemctl restart rsyslog

[Нужен скриншот: перезапущенного rsyslog (systemctl status rsyslog)]

Клиенты rsyslog (HQ-RTR, BR-RTR, BR-SRV):

dnf install -y rsyslog

nano /etc/rsyslog.conf

*.warning @@192.168.100.2:514

[Нужен скриншот: конфигурации клиента rsyslog (*.warning @@192.168.100.2:514)]

systemctl restart rsyslog

На HQ-SRV не должно быть строки с @@192.168.100.2:514, потому что в задании указано что сервер не должен сам себе отправлять логи.

Ротация логов на HQ-SRV:

nano /etc/logrotate.d/opt-logs

/opt/*/*.log { weekly rotate 4 compress missingok notifempty minsize 10M sharedscripts postrotate /bin/kill -HUP `cat /var/run/syslogd.pid 2>/dev/null` 2>/dev/null || true endscript }

[Нужен скриншот: HQ-SRV: ls /opt/ — папки hq-rtr, br-rtr, br-srv с лог-файлами]
